# 数据结构学习记录

## 项目简介
本项目记录了我在学习电子科技大学的数据结构课程过程中的学习笔记、实验代码和总结。
本项目为初学者提供了系统地学习线性表、树、图、查找算法和排序算法等基础知识的相关资料。

## 学习内容

### 1. 线性表
- **顺序表与单链表**：实现了顺序表和单链表的基本操作，包括初始化、插入、删除、查找和遍历。
- **实验代码**：[顺序表与单链表操作](https://github.com/yourusername/data-structures/tree/main/SequentialList)

### 2. 树与二叉树
- **二叉排序树（BST）**：实现了BST的创建、查找、插入和删除操作。
- **AVL树**：学习了AVL树的平衡旋转操作，并实现了相关算法。
- **哈夫曼树与哈夫曼编码**：实现了哈夫曼树的构建和哈夫曼编码的生成。
- **实验代码**：[二叉树操作](https://github.com/yourusername/data-structures/tree/main/BinaryTree)

### 3. 查找算法
- **顺序查找、折半查找和分块查找**：分析了不同查找算法的时间复杂度和适用场景。
- **哈希表**：学习了哈希表的构造方法和冲突处理机制。
- **实验代码**：[查找算法实现](https://github.com/yourusername/data-structures/tree/main/SearchAlgorithms)

### 4. 排序算法
- **插入排序、冒泡排序、选择排序**：实现了基本的排序算法，并分析了它们的时间复杂度。
- **快速排序、归并排序和堆排序**：学习了高效的排序算法，并通过实验验证了它们的性能。
- **实验代码**：[排序算法实现](https://github.com/yourusername/data-structures/tree/main/SortingAlgorithms)

### 5. 图
- **图的存储结构**：邻接矩阵和邻接表的实现。
- **图的遍历**：深度优先搜索（DFS）和广度优先搜索（BFS）。
- **最小生成树**：Prim算法和Kruskal算法的实现。
- **最短路径**：Dijkstra算法和Floyd算法的实现。
- **实验代码**：[图操作](https://github.com/yourusername/data-structures/tree/main/Graph)

## 学习总结
通过本次学习，我掌握了数据结构的基本概念和常用算法，并通过实验加深了对理论知识的理解。数据结构不仅是计算机科学的基础，也是解决实际问题的重要工具。在后续学习中，我将继续深入研究数据结构的高级应用，并探索其在不同领域的实际应用。

## 使用说明
- **环境要求**：C语言开发环境 例如VS2019
- **运行代码**：将代码文件保存到本地，使用C语言编译器编译并运行。
