# 数据结构学习记录

## 项目简介
本项目记录了我在学习电子科技大学的数据结构课程过程中的学习笔记、实验代码和总结。
本项目为初学者提供了系统地学习线性表、树、图、查找算法和排序算法等基础知识的相关资料。

## 项目结构
```
data-structures/
├── Class_Experience/          # 课堂实验相关代码和文档
├── LinkList/                  # 单链表的实现
├── DuList/                    # 双向链表的实现
├── SqStack/                   # 顺序栈的实现
├── SqQueue/                   # 顺序队列的实现
├── Sqlist/                    # 顺序表的实现
├── LinkStack/                 # 链式栈的实现
├── LinkQueue/                 # 链式队列的实现
├── Tree/                      # 树的实现（二叉树、二叉排序树）
├── Graph/                     # 图的实现（包括邻接矩阵和邻接表）
├── Search/                    # 查找算法（顺序查找、折半查找）
├── test/                      # 测试代码和示例
├── Algorithm.pdf              # 算法相关学习资料
├── DataStructure.pdf          # 数据结构相关学习资料
└── README.md                  # 项目说明文件
```

## 使用说明
- **环境要求**：C语言开发环境 例如VS2019
- **运行代码**：将代码文件保存到本地，使用C语言编译器编译并运行。
